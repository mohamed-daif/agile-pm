# Plugin System Documentation

> Build custom integrations for Agile-PM

## Overview

The Agile-PM plugin system allows you to:

- Integrate with external services (GitHub, Jira, Slack, etc.)
- React to system events (task created, completed, etc.)
- Extend functionality without modifying core code
